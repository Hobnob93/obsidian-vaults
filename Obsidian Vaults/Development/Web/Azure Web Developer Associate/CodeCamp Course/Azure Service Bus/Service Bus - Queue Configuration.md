#azure #az-204 

When creating a queue, there are a few configuration options:
- **Name**
	- the name of the queue
- **Max Queue Size**
	- the max size of queue in GB
- **Max delivery count**
- **Message TTL**
	- TTL: time to live
	- can set days, hours, minutes, seconds
- **Lock Duration**
	- can set days, hours, minutes, seconds
- **Dead lettering**
	- when messages expire
- **Partitioning**
